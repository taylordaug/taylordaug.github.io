use a loop if you want to repeat something over and over. loops check for a condition. if it returns true, a code block will run. then the condition will be checked again and if it still returns true, the code block will run again. it repeats until the condition returns false. make sure your loop knows when to stop, otherwise it will run forever and break your program.
  • for...in executes code once for each element in expression
    •       for variable in expression do
        code
      end
  • while executes code while conditional is true
    •       while conditional do
        code
      end
  • until executes code while conditional is false
    •       until conditional do
        code
      end
  • next jumps to next iteration of the most internal loop
    •       for i in 0..5
        if i < 2 then
          next
        end
          puts i
      end

      => 2
      => 3
      => 4
      => 5
  • redo restarts the iteration of the most internal loop, without checking loop condition
    •       for i in 0..5
        if i < 2 then
          puts i
          redo
        end
      end

      => 0 (infinite loop)
  • break terminates the most internal loop
    •       for i in 0..5
        if i > 2 then
          break
        end
          puts i
      end

      => 0
      => 1
      => 2
  • for repeats until condition evaluates to false
  • use a counter as a condition: (var i = 0; i < array.length; i++)
    •     for ([initialize]; [condition]; [increment]) {
        statement;
    }
  • for...in creates a loop iterating over property names
    •     for (variable in object) {
        statement;
    }
  • for...of creates a loop iterating over property values
    •     for (variable of object) {
        statement;
    }
  • while executes statements as long as specific condition evalutes to true
    •     while (condition)
      statement
  • do...while repeates until a specified condition evaluates to false
    •     do
      statement
    while (condition);
  • label provides a statement with an identifier so it can be used elsewhere in your program
    •     label :
      statement
  • continue used to restart a while, do-while, for, or label statement
    •     continue;
  • break terminates the most internal loop
    •     for (i = 0; i < a.length; i++) {
      if (a[i] == theValue) {
        break;
      }
    }
